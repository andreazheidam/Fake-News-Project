Be sure to install the modules in the requirements.txt 

Before running the code, you should insert file pathes in the first cell for:
    where you have locally saved the complete fake news data
    where you want to save a local copy of the cleaned fake news data
    where you have locally saved the BBC news extension data

There is a folder called observations, filled with python scripts used to find interresting patterns about the fake news data
To use this file, insert the path of your fake news data in the first code cell
